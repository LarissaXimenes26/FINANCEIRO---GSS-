Como usar
  • Exporte as planilhas do OMIE em formato .xlsx
  • Importe cada planilha na aba correspondente — as abas Matriz/Filial são detectadas automaticamente
  • Ao importar um novo arquivo, apenas aquele módulo é atualizado; os demais permanecem intactos
  • Os dados são salvos no navegador (localStorage) e restaurados automaticamente ao reabrir
  • Use "Limpar dados" para remover um módulo individualmente sem afetar os outros
